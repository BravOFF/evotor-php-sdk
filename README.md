# evotor-php-sdk

!!ПОКА НЕ ДЛЯ ИСПОЛЬЗОВАНИЯ В PRODUCTION!!

API клиент для облака Evotor

Установка
------------

Рекомендуемый способ установки через
[Composer](http://getcomposer.org):

```
$ composer require kilylabs/evotor-php-sdk
```

Использование
-----
#### Инициализация
```php
<?php

require 'vendor/autoload.php';

```
